DHIS2 Sumarised API Documentation

This is a summarised list of all endpoints available on DHIS2, version 2.30. The endpoints can be accessed in the two listed formats, JSON and XML.
Please note that authentication is not handled here, and will need to be configured before any of these are accessible.

This documentation is not comprehensive in any way. Kindly refer to the official DHIS2 documentation for complete reference of all the functions and capabilities of the system.
